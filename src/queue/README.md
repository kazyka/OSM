# A Heap-Based Priority Queue

To compile, type:

    make

To run, type:

    ./bouncer

To run with Valgrind, type:

    make valgrind

To run with Valgrind with the input in `input.txt`:

    make quickgrind

To debug with GDB (with the input from `input.txt` and GDB script `gdb.script`):

    make debug
